# 基于 SCCT 理论的自然灾害微博舆情生成式回应提示词模板

## 项目简介

本项目构建了一套**面向自然灾害情境的生成式人工智能微博舆情回应提示词模板**，以台风“摩羯”事件为研究对象，融合**情境危机沟通理论（Situational Crisis Communication Theory, SCCT）**、公众言语行为分类与主题识别机制，用于规范和约束大模型在突发灾害舆情中的回应生成。

该提示词体系旨在缓解生成式人工智能在灾害舆情回应中存在的策略模糊、情绪失配与信息回应不足等问题，为相关实证研究与模型对比实验提供可复现、可解释的标准化输入。

---

## 研究目标

- 构建符合危机沟通理论的生成式人工智能回复约束机制  
- 实现公众评论**言语行为 × 主题类型 × 灾害阶段**的策略匹配  
- 为生成式人工智能灾害舆情回应效果评估提供统一的提示词模板  
- 提升灾害舆情研究中方法设计的透明性与可复现性  

---

## 理论基础

- **情境危机沟通理论（SCCT）**  
  用于指导不同危机责任归因情境下的回应策略选择，包括否认型、淡化型、支持型与重塑型策略。

- **言语行为理论（Speech Act Theory）**  
  用于区分公众评论的交互意图，如提问、表达观点、经历分享与请求帮助。

---

## 使用方法

1. **输入要素**
   - 微博评论文本  
   - 评论类型（人工标注或模型识别）  
   - 主题类型  
   - 灾害阶段（潜伏期 / 爆发期 / 成熟期 / 衰退期）

2. **调用方式**
   - 可作为本地部署大模型（如 GLM、LLaMA 等）的系统提示词或角色提示词  
   - 适用于生成式人工智能灾害舆情回应生成实验

3. **输出结果**
   - 回应结构清晰、策略明确  
   - 同时兼顾权威信息回应与情感支持  
   - 符合危机沟通理论与灾害风险沟通规范

---

## 应用场景

- 自然灾害舆情自动回应生成研究  
- SCCT 理论与生成式人工智能融合的实证分析  
- 不同回复策略的效果评估（如情感支持、信息质量等指标）  
- 政务新媒体或应急管理智能辅助系统的原型设计

---

## 注意事项

- 本提示词模板主要用于学术研究与实验分析，不直接替代真实应急管理决策  
- 示例回复中涉及的“信息检索”需结合实际权威数据源实现  
- 不同大模型对提示词的响应存在差异，建议根据模型特性进行适配性调整
